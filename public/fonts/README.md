# Gelica font

Place Gelica font files here for the app to use:

- `Gelica-Regular.woff2` (or .woff)
- `Gelica-SemiBold.woff2` (optional, for bold headings/buttons)

**Download:** FontSpace, Fontspring demo, or purchase from Eclectotype.

If files are missing, the app falls back to Georgia for headings/buttons.
